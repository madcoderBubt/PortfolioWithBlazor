# Portfolio .NET

A personal portfolio website built with ASP.NET Core.

## Features

- **Modern UI**: Clean and responsive design with a focus on aesthetics.
- **Dark/Light Mode**: Toggle between themes for comfortable viewing.
- **Glassmorphism**: Stylish header and card effects.
- **Responsive**: Optimized for both desktop and mobile devices.

## Tech Stack

- **Framework**: ASP.NET Core 8.0 (Razor Pages / MVC)
- **Styling**: Vanilla CSS with custom variables and modern layout techniques (Flexbox/Grid).
- **Icons**: FontAwesome (or similar library if used).

## Getting Started

### Prerequisites

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0)

### Running Locally

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd "portfolio dotNet"
    ```
3.  Run the application:
    ```bash
    dotnet watch run
    ```
4.  Open your browser and visit `http://localhost:5104` (or the port shown in your terminal).

## Project Structure

- `wwwroot/`: Static assets (CSS, JS, Images).
- `Pages/`: Razor pages for the application.
- `Components/`: Reusable UI components.
- `Models/`: Data models.
- `Services/`: Business logic and data services.
